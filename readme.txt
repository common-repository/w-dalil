This Plugin makes a phonebook provide Many for you , with the ability to list all the categories and cities you have . custom search included 

== Description ==

a Phonebook , this plugin make custom post type for you (Dalil) and with it you can add items include the phone number, email, website,Location of course .

* well desgined with bootstrap and fontawesome 
* full responsive with all widths
* custom shortcodes for listing cities, categories, show the items and a shortcode for the search (it must be in same page)
* clicking on the website will open it with new window
* none of your information on the webpage is able to copy with Ctrl+C or with right click 
* click on the email will pop a window with inputs name, subject and a message will send automatically.

pop with google map will be added soon .

* you can extract your data with a click to .xml file 
* import is easy and the same . ( xml with information like the provided in the exported one ) 
* there is custom shortcode for end-front users to add an item to the dalil ,not published till accept.

this plugin built for english and arabic readers (Turkish soon)


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/w-dalil` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add your inforamtion.
4. create your main page and add the [FW-dalil],you can add the categories lists in your sidebar with shorcode [list-cat-dalil] and the cities with [list-cities-dalil]
5. thats it . nothing complicated
